Recommended Reading Path

The chapters are independent, but this order builds the most useful mental model: estimator contracts first, image task contracts second, learned tensor operations third, language representation and generation fourth, production operation fifth.

Basic MLLearn feature matrices, supervised and unsupervised algorithms, validation, metrics, and classical model behavior.
Computer Vision MLMove from general ML metrics to image-specific tasks: filters, classification, detection, masks, augmentation, and visual evaluation.
Deep LearningStudy CNN mechanics, tensor shape contracts, Torch base layers, normalization, attention, and layer-level simulations.
NLP and TransformersMove from count vectorizers, TF-IDF, and Naive Bayes to embeddings, attention, decoding, retrieval, and LLM evaluation.
MLOps and Production MLConnect models to labels, training, evaluation, deployment, monitoring, feedback loops, large-model operations, and agent safety.
Future ChaptersAdd recommender systems, time series, optimization, data engineering, and advanced evaluation as the book grows.
